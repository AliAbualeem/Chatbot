AI Chatbot Integration using Next.js & OpenRouter
This project demonstrates the integration of an AI-powered chatbot into a web application using Next.js, HTML, and CSS, with OpenRouter.com as the AI API provider.

 Features:

Built with Next.js for fast and modern web development
Clean UI using HTML & CSS
Real-time chatbot interaction
Uses OpenRouter API to access free AI models
Easy to customize and extend

 Technologies Used:
Next.js
HTML
CSS
OpenRouter API
